Stacks

만들고자 하는 목표에 대해 필요한 기술들을 빠르게 습득하고 구현합니다.
탄탄한 기본기를 기반으로 언제든 새로운 기술에 적응하겠습니다.
기술 구분 로고 활용 수준
JAVA Java에서 지원하는 Swing으로 도서관리 프로그램을 만들었습니다.
이과정에서 객체지향, 예외, 네트워크, 입출력, 컬렉션 등에 복습하였습니다.
JSP 웹의 기본 동작원리를 학습했습니다.
DAta의 이동, session과 cookie , 요청과 응답, MVC모델1 을 고려하며 구현했습니다.
Spring MVC모델2 를 활용하여 유지보수와 확장성이 뛰어난 코드를 학습했습니다.
Mybatis, loombok 등으로 DB와 연결합니다.
DI를 활용하여 의존하는 객체를주입하고 유연한 코드를 구현했습니다.
HTML5 & CSS3 웹의 기본이 되는 언어들로 두개의 WEb프로젝트를 개발하였습니다.
HTML태그를 적재적소에 위치하고,CSS로 사용자에게 부각시킵니다.
브라우저 크기에 반응하고 사용자 입력값에 반응하기위해 Javascript와도 융합합니다.
JavaScript HTML의 기본 동작제어와 CSS를 커버하는 스타일링을 기본적으로 다룹니다.
필요시 입력값 검사와 페이지 생성,이동 AJAX를 통한 비동기 통신 등 무궁무진한 언어라고 생각합니다.
DataBase SQL의 기본 권한,명령어를 프로젝트에 적용했습니다.
Left outer join,   union all,   Rollup,   rollback,   Rank(),   Level(),   partition 등을 학습했습니다.
객관화하기 어려운 관계로, 제가 이해하고 구현할 수 있는 수준을 작성했습니다.
